# 更新记录 / Changelog

本项目版本号遵循 `主.次.修订`。日期为发布当天（本地时区）。

## [0.2.1] - 2026-06-23

### 修复

- **打包版偶发「后台未启动」**：根因是前端与后端 sidecar 的启动竞态 + 端口缓存错误，而非后端真的没起来。
  - 打包版后端走动态端口（`TASKDECK_PORT=0`，由系统分配），Rust 需等 node sidecar 冷启动并上报 `TASKDECK_PORT=<n>` 后才知道实际端口。
  - 旧逻辑：webview 加载快于 sidecar 冷启动，抢先 `invoke("server_port")` 拿到 `null`，却用 `_basePromise ??=` 把这个坏基址（`http://127.0.0.1:null`）**永久缓存**。此后 `waitForHealth` 15 秒内反复请求同一个坏地址，必然超时 → 显示「后台未启动」。表现为时好时坏，取决于谁先就绪。
  - 修复（`app/frontend/src/api/client.ts`）：端口未就绪时 `resolveBase` 返回 `null` 且**不缓存**；`waitForHealth` 每轮重新解析端口，直到拿到真实端口才落定基址并通过健康检查。

## [0.2.0] - 2026-06-22

### 新增（M2 可用性增强）

- **优先级**：四档 急 / 高 / 低 / 中，各有点缀色。
- **已完成任务**：全部任务页「已完成」浮窗集中查看；完成满 7 天物理删除。
- **日历**：年 / 月可点击微调与直接输入切换；某天圆点按当天最高优先级着色。
- **标签库**：标签升级为可管理字典表（含预设），支持新增 / 删除（二次确认）；AI 打标签优先复用标签库。砍掉「分类」维度，只保留标签。
- **设置与模型**：侧栏齿轮进设置，配置 DeepSeek（OpenAI 兼容），模型下拉选 V4 Flash / V4 Pro；侧栏底部常显当前接入模型。
- **发布版瘦身**：剔除 Claude Agent SDK 及其原生二进制（约 219MB），发布版仅走 DeepSeek（纯 `fetch`）；sidecar 改用官方标准 node 运行时。

## [0.1.0] - 2026-06-22

### 新增（M1 闭环 MVP）

- 自然语言输入 → AI 解析（打标签 / 推断日期 / 定优先级）→ 入库 → 四视图（对话 / 日历 / 标签 / 全部）端到端跑通。
- 技术栈：Tauri 2 + React 19 + Vite；Node + Express + better-sqlite3；本地 SQLite（WAL）。
