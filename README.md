# 万事 TaskDeck

> 一句话登记任务，剩下交给 AI。个人任务中控台桌面应用。

输入一句自然语言（如「下周三前交季度报告」），后台 AI 自动完成**打标签、推断日期、定优先级**，并整理进日历与标签视图。AI 走 **DeepSeek**（OpenAI 兼容，在设置里填 API Key 即可，模型可选 V4 Flash / V4 Pro）。数据全部存本地 SQLite，私有、自用。

## 截图

对话区（ChatGPT 式）输入任务 → AI 即时解析为结构化任务卡：

- 对话 / 日历日程 / 按标签 / 全部任务，四视图左侧切换
- Apple 视觉基底 + ChatGPT 式对话交互

## 技术栈

- **桌面壳**：Tauri 2 (Rust)
- **前端**：React 19 + TypeScript + Vite
- **后端**：Node + Express + better-sqlite3；AI 走 DeepSeek（OpenAI 兼容，纯 `fetch`，无第三方 SDK）
- **数据**：本地 SQLite（WAL）

## 快速开始

> 前置：Node ≥ 20、Rust 工具链（Tauri 需要）、macOS Xcode CLT；以及一个 DeepSeek API Key（首次启动时在引导里填）。

```bash
cd app

# 方式一：完整桌面应用（Tauri 窗口）
npm run dev

# 方式二：仅浏览器调试（不编译 Rust，最快）
npm run dev:browser   # 然后浏览器打开 http://localhost:5173
```

首次 `npm run dev` 会编译 Tauri（Rust，稍慢）；之后秒起。

## 当前进度

**M1 闭环 MVP 已完成**：输入 → AI 分析 → 入库 → 四视图呈现，端到端跑通。

**M2 可用性增强（进行中）**，已落地：

- **优先级**：四档 急 / 高 / 低 / 中（`urgent/high/medium/low`），各有点缀色。
- **已完成任务**：全部任务页右下角「已完成」浮窗集中查看；任务划掉当次仍留主列表，下次打开自动归入浮窗；完成满 **7 天物理删除**；弹窗底部「**清理所有已完成任务**」可一键归档全部（二次确认）。
- **日历**：左上角年 / 月支持点击微调与直接输入数字切换；某天的圆点按**当天最高优先级**着色。
- **标签库**：标签升级为可管理的字典表（预设 学习/工作/生活/娱乐/健康/财务/家庭/社交/旅行），标签页支持新增、删除（删除需二次确认，防误删）；AI 打标签时优先复用标签库，新标签自动并入。任务模型已**砍掉「分类」维度，只保留更灵活的标签**。
- **设置与模型**：侧栏齿轮进设置，配置 DeepSeek（OpenAI 兼容，需 API Key），模型下拉选 V4 Flash / V4 Pro；侧栏底部常显当前接入模型。
- **首启引导**：第一次启动**强制先配置 DeepSeek**（填 API Key，未填时「开始使用」禁用）才能进入主界面；配置写入本机 `settings`（`setup_done` 标记），之后不再出现，仍可在设置里随时调整。
- **品牌与视觉**：自绘 App 图标（任务卡叠加 + 蓝色对勾），`tauri icon` 生成 macOS/iOS/Android 全套；Web favicon 与窗口标题同步为「万事 TaskDeck」。文案精简为「你说一句，剩下的交给我 / 输入任务，我来整理」。
- **对话页顶部 DDL 时间轴**：近 3 天有截止时间的任务按时刻横排成一条可滑动的时间轴（有任务按宽度撑开、空白处最小间距，跨天插日期刻度），点任务上的圆圈即完成；右上角三角收放，无任务默认收起、有任务默认展开。卡片为独立不透明浮层。
- **标签排序**：标签页按任务数量降序展示，数量相同按拼音首字母升序。
- **悬浮窗（桌面悬浮球）**：设置弹窗里开关「悬浮窗」（**默认开**），桌面常驻一颗圆形悬浮球。**点击**展开为「近期 / 全部」任务卡片（可勾选完成、⤢ 唤起主窗口），**拖动**换位；拖到**左 / 右边缘**自动吸附收成球、拖到**中间**为自由浮窗；卡片「–」最小化就近吸边收球。窗口**置顶 + 跨工作区可见**，独立 Vite 入口、共享同一后端。拖动用手动 `setPosition`（保留真实松手判定）。交互机制全文档见 `~/Projects/Doc/floating-window-interaction.md`。
- **迫近高亮（0.3.0）**：DDL 前 2 小时内（含已过期未完成）任务标题实时变红——覆盖悬浮窗、主窗口各页与对话页时间轴；有迫近任务时悬浮球白光晕脉动 + 透明度闪烁 + 浮动。**重要度（静态优先级）与紧急度（实时派生）分离，互不污染**。
- **优先级随 DDL 自动升级（0.3.0）**：后端定时器按距截止远近自动提升开放任务优先级（≤2h→急、≤24h→高、≤72h→中，只升不降、幂等）。
- **相对时间解析（0.3.0）**：支持「十分钟后 / 半小时后 / 2 小时后」等相对说法，给 AI 注入当前精确时刻、换算为绝对截止时刻并跨零点进位。
- **全局复制 & 日历下拉（0.3.0）**：框选复制只取文字、不带布局空白；日历年 / 月改为下拉选择（年份今年 ±10），选即切。
- **透明窗黑底已修复（0.3.0）**：原 M5「macOS Sequoia 透明窗四角黑底」遗留，已用常驻 `filter` 动画持续重绘解决，静止 / 告警两态均不黑。

## 接入说明

AI 调用通过 `app/server/src/ai/provider.ts` 抽象，当前**唯一 provider 为 DeepSeek**：

- 设置里填 DeepSeek API Key，走 `openaiCompatProvider`（OpenAI 兼容，默认 `https://api.deepseek.com`，纯 `fetch`、无第三方 SDK）。可选模型为当前生产版 `deepseek-v4-flash` / `deepseek-v4-pro`；旧别名 `deepseek-chat` / `deepseek-reasoner` 官方将于 2026/07/24 停用，故未列出。
- 未配置 Key 时，路由层 `fallbackAnalysis` 兜底降级入库（仍记下原文，不丢任务）。

> 早期版本曾支持「本机 Claude Code（Agent SDK）」作为默认零配置通路，后已**整体下线、统一走 DeepSeek**，`server` 不再依赖 `@anthropic-ai`。相关瘦身历史见 `doc/milestones.md` M3。
